# 航班卡片显示约束

## 1. 特价标签 (Special Badge)
- **触发条件**：折扣 >= 40%
- **位置**：`absolute top-4 right-4`
- **样式**：`bg-gradient-to-r from-orange-500 to-red-500`，白色文字，圆角full
- **内容**：`<i class="fas fa-bolt mr-1"></i>特价`

## 2. 折扣标签 (Discount Badge)
- **触发条件**：折扣 >= 30%

### 40%+ 航班（特价）
- **位置**：卡片右上角，特价标签左侧（`absolute top-4 right-20`），与特价标签同一水平线
- **样式**：`absolute`，`bg-green-100 text-green-800`，圆角，阴影，`z-10`
- **内容**：`-${discount}%`

### 30-39% 航班（非特价）
- **位置**：价格区域内，`text-right` 容器内，价格上方独立一行
- **样式**：`inline-block bg-green-100 text-green-800`，圆角
- **内容**：`-${discount}%`

## 3. AI推荐标签 (AI Recommendation Badge)
- **触发条件**：推荐分数 >= 70
- **位置**：价格区域内的"查看详情"按钮正上方，居中
- **样式**：`flex justify-center`，`bg-gradient-to-r from-purple-500 to-indigo-500`，白色文字，圆角full
- **内容**：`<i class="fas fa-robot mr-1"></i>AI推荐 ${score}分`

## 4. 反馈按钮 (Feedback Buttons)
- **触发条件**：每5个卡片显示一次（第5、10、15...个）
- **位置**："查看详情"按钮下方，价格区域内
- **样式**：`mt-3 pt-3 border-t border-gray-200`
- **内容**：有用/无用按钮

## 5. 标签层级关系
```
卡片右上角（absolute）:
  right-4: 特价标签 (40%+)
  right-20: 折扣标签 (40%+，特价左边)

卡片内容区（价格区域）:
  - 折扣标签 (30-39%，价格上方)
  - 价格信息
  - AI推荐标签 (查看详情按钮上方，居中)
  - 查看详情按钮
  - 收藏按钮
  - 反馈按钮 (每5个显示一次)
```

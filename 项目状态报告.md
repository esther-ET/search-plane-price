## 一、项目概述

**产品定位**：聚合各大OTA平台和航司特价机票信息的智能发现平台，通过AI技术帮助用户高效识别高性价比航班。

**核心价值**：
- 信息聚合：一站式整合分散的特价机票信息
- 智能筛选：AI算法识别真正的高性价比航班  
- 规则简化：将复杂的促销规则转化为易懂的优惠信息
- 个性化推荐：根据用户偏好和历史行为推荐合适航班

## 三、技术架构

### 3.1 前端技术栈
- **框架**：纯HTML/CSS/JavaScript（无框架依赖）
- **UI库**：Tailwind CSS 3.x（CDN方式）
- **图标**：Font Awesome 6.4.0
- **JavaScript**：ES6+，模块化组织

### 3.2 项目结构
```
price-demo-3hour/
├── index.html              # 首页（集成AI推荐）
├── search.html            # 搜索列表页
├── detail.html            # 航班详情页
├── preferences.html       # 用户偏好设置页
├── css/                   # 样式文件（暂无）
├── js/                    # JavaScript模块
│   ├── data.js           # 模拟数据系统
│   ├── home.js           # 首页逻辑（AI推荐集成）
│   ├── search.js         # 搜索页逻辑
│   ├── detail.js         # 详情页逻辑
│   ├── preferences.js    # 用户偏好管理
│   ├── recommendation.js # AI推荐算法引擎
│   ├── feedback.js       # 用户反馈系统
│   └── abtest.js         # A/B测试框架
├── assets/               # 静态资源
```

## 七、技术债与注意事项

### 技术债
1. **硬编码数据**：目前使用静态模拟数据，需要转换为动态API
2. **无后端服务**：所有逻辑在前端，限制功能扩展
3. **SEO优化不足**：纯前端渲染，SEO效果有限
4. **性能优化**：大规模数据时可能存在性能问题

### 安全注意事项
1. **模拟数据**：当前为演示数据，不能用于真实预订
2. **API安全**：未来接入真实API时需要考虑认证和限流
3. **用户隐私**：如收集用户偏好，需要隐私政策说明
